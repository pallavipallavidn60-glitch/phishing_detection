# 📌 About The Project

The **Phishing Detection System** is a cybersecurity-based web application that helps users identify whether a website URL is **safe or phishing**.

This project is designed to improve online safety by detecting suspicious links using intelligent analysis techniques.

It helps users:
- 🔍 Detect phishing websites
- 🛡️ Improve cybersecurity awareness
- ⚡ Analyze suspicious URLs quickly
- 🌐 Stay protected from online scams

# ⚙️ Installation

## Step 1: Clone Repository

```bash
git clone https://github.com/pallavipallavidn60-glitch/phishing_detection.git
```

## Step 2: Open Project Folder

```bash
cd phishing_detection
```

## Step 3: Install Requirements

```bash
pip install -r requirements.txt
```

## Step 4: Run Application

```bash
python app.py
```

# 🌐 Usage

1. Enter a website URL
2. Click on Detect
3. The system analyzes the URL
4. Displays whether the link is Safe or Phishing
